Masiva is a flexible Digital Asset Management system built around a set of core Open Source tools:

Grails : User Interface and Application Framework
Mongodb : Metadata and file storage (GridFS)
RabbitMQ : Message Queueing
Apache Solr : Faceted Search Engine
Spring Security : Flexible I&AM management
FFMPEG : Simple video transcoding and thumbnail generation
JAI, Exif & XMP Libraries : Simple image manipulation and metadata extraction
jQuery & jQuery UI : Javascript library
It has been designed to address some key concerns the author has with existing proprietary Digital Asset Management solutions, including unnecessary complexity, inflexibility, massive upfront cost and steep learning curve, legacy of proprietary additions and restrictions.
